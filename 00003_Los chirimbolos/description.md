|Operación	|Operando | | | | | | | | | | | | |
|-------------	|----------	|---	|---	|---	|---	|---	|---	|---	|---	|---	|---	|---	|---	|
| Suma | + | | | | | | | | | | | | |
| Resta | - | | | | | | | | | | | | |
| Multiplicación | * | | | | | | | | | | | | |
| División | / | | | | | | | | | | | | |
 
 
>  :thinking_face:  Si por ejemplo tenemos la siguiente expresión matemática...
>
> ```python
> 5 + 30 * 20
> ```
>
> y esta otra..
>
> ``` python
> (5 + 30) * 20
> ```
